# COSC_310 ByteMe-Project
> Group name: Byte Me
> 
> Cohen Kucher (37980372) , Gulsher Assad (23641871), Manpreet Kaur (33647884), Zhenyu Li（99978793）
> 
> Browse movies, read/write reviews and ratings, via a clean REST API and file-based storage (JSON/CSV). Built for UBCO COSC 310.

- **Dataset**: IMDB User Reviews -- https://www.kaggle.com/datasets/sadmadlad/imdb-user-reviews

---


## Features

- **Accounts**: register & login (secure password hashing).
- **Movies**: browse/search by title, genre, or keywords; sorting & basic filters.
- **Ratings**: 1–10 scale (0.5 step); compute and display per-movie average.
- **Reviews**: create/edit/delete your own; view others with timestamps and view counts.
- **Comments**: threaded replies under a review.
- **Moderation**: report content; local block; admin review & removal.
- **Storage**: JSON/CSV files (no SQL database), portable and easy to inspect.
